# TimeQuest Log

## Overview

TimeQuest Log is a comprehensive gaming tracker that enables users to monitor their gaming hours and sessions by referencing the executable files used to launch games. The application integrates with Twitch's IGDB API to match users' games with a database of known titles, providing details such as official game titles, genres, and release dates. Users can create accounts to securely access their personalized gaming statistics across both desktop and web platforms.
